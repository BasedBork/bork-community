# Core Concepts

The fundamentals that make you effective with Claude Code.

## Overview

Claude Code is more than a chatbot in your terminal. Understanding these core concepts helps you get real work done.

## Concepts

| Concept | What It Is | Why It Matters |
|---------|------------|----------------|
| [Plan Mode](./PLAN_MODE.md) | Structured approach for complex tasks | Prevents Claude from diving in without thinking |
| [Slash Commands](./SLASH_COMMANDS.md) | Built-in commands like `/clear`, `/undo` | Quick actions without natural language |
| [Context Management](./CONTEXT_MANAGEMENT.md) | Controlling what Claude knows | Keeps Claude focused and accurate |
| [Skip Permissions](./SKIP_PERMISSIONS.md) | Running without approval prompts | Faster workflow when appropriate |

## Quick Reference

### Most Used Commands

```
/clear    - Reset context, start fresh
/undo     - Undo last change
/help     - Show all commands
/model    - Switch Claude model
/stats    - Show token usage
```

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Shift+Tab` | Enter plan mode |
| `Ctrl+C` | Cancel current operation |
| `Tab` | Accept suggestion |
| `Up/Down` | Navigate history |

### Mental Model

Think of Claude Code in three modes:

1. **Chat Mode**: Ask questions, get answers
2. **Edit Mode**: Claude makes changes to files
3. **Plan Mode**: Claude thinks before acting

Start in chat mode. Use plan mode for anything non-trivial.
