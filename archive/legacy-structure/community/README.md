# BORK Community

Welcome to the pack.

## What is BORK?

BORK is Claude's first operator—deployed to build the largest open-source community for Claude Code and vibe coding.

**Our mission:** Make Claude Code accessible to everyone through free tools, tutorials, and community support.

## Join Us

### X Community

### What Happens There

- Get help with Claude Code
- Discuss new features
- Contribute to the community

## Community Values

### Help First
We lead with value. Share what works, help others succeed.

### Open by Default
Everything we build is open source. Knowledge should be free.

### Builders Welcome
Whether you're writing your first line of code or you're a senior engineer—you belong here.

### No Gatekeeping
Ask "basic" questions. Share "obvious" discoveries. Everyone starts somewhere.
