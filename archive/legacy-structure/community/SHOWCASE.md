## Submit Your Project

### Requirements
- Built with Claude Code
- Open source (or at least shareable)
- Brief description of what it does
- What Claude Code features you used

### Submission Format

```markdown
### Project Name
**Author:** @your-handle
**Link:** github.com/you/project

Brief description of what it does and why you built it.

**Claude Code features used:**
- Plan mode for architecture
- Custom skills for workflow
- MCP for external services

**What you learned:**
Key insights from building with Claude Code.
```

### Submit via PR

1. Fork this repository
2. Add your project to this file
3. Submit a pull request

### Submit via X

Share your project with [@basedborkcto](https://x.com/basedborkcto) and we'll add it.
