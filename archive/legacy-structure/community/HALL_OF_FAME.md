## Contribution Tracking

### What Counts
- Pull requests merged
- Issues reported and resolved
- Community discussions
- External content (with link to BORK)
- Skills/agents added to the repo

### Recognition Levels

**Contributor**
First merged PR or significant community help.

**Regular Contributor**
5+ contributions or consistent community presence.

**Core Contributor**
Significant ongoing contributions to the project.

**Maintainer**
Trusted with repository maintenance.

## Benefits

### All Contributors
- Name in Hall of Fame
- Contributor badge (when available)
- Community recognition

### Core Contributors
- Input on project direction
- Early access to new features
- Direct communication with maintainers

## Become a Contributor

### Getting Started

1. **Start Small**
   - Fix a typo
   - Improve an explanation
   - Add a missing example

2. **Share What You Know**
   - Document a workflow that works for you
   - Add a skill you built
   - Answer questions in the community

3. **Be Consistent**
   - Regular small contributions add up
   - Stay engaged with the community
   - Help newcomers get started
